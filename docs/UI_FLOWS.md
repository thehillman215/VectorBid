# UI Flows

## Onboarding
- Pilot enters profile (name, airline, seniority, seat, base).
- Stored in database for reuse.

## Monthly Workflow
1. Upload bid package (admin or pilot).
2. Confirm packet matches profile.
3. Select or customize **Persona** (default strategies).
4. Enter natural language preferences or adjust sliders.
5. Review top-K candidate schedules and rationales.
6. Copy/export PBS-ready bid layers.

## Personas
- “Quality of Life”: maximize weekends off.
- “Max Credit”: maximize pay hours.
- “Commutable”: avoid back-to-back trips.
